SCPUpdateReaderDelegate

@protocol SCPUpdateReaderDelegate <NSObject>

Implement this protocol to handle events when updating a reader.

  • -terminal:readerUpdateAvailable:installUpdate:

    An update is available to be installed on the connected reader. You should notify your app’s user that an update is available, and ask if they want to proceed with the update.

    During an update, the reader becomes unusable. You may use the didReportReaderUpdateProgress delegate method to follow the progress of the update. After the update completes, the didCompleteReaderUpdate delegate method will be called.

    Declaration

    Objective-C

    - (void)terminal:(nonnull SCPTerminal *)terminal
    readerUpdateAvailable:(nonnull SCPReaderUpdate *)readerUpdate
            installUpdate:
                (nonnull SCPInstallUpdateCompletionBlock)installUpdate;

    Swift

    func terminal(_ terminal: Terminal, readerUpdateAvailable readerUpdate: Any!, installUpdate: @escaping InstallUpdateCompletionBlock)

    Parameters

    terminal

    The originating terminal.
    readerUpdate

    An object containing information about the update.
    installUpdate

    Call this block with a boolean value indicating whether you want to proceed with the update. Calling installUpdate(true) begins installing the udpate.
